# 3. Protocolo de Triangulação de Dados e Auditoria

## Metodologia de Consenso Auditável
Para garantir a cientificidade do método **TBR-2D-BO**, estabelece-se um protocolo de consenso baseado em métricas estatísticas de concordância, indo além de simples reuniões consultivas.

## Fluxo de Trabalho
1.  **Análise Cega Individual:** Três especialistas codificam os itens independentemente usando o Guia de Codificação (Etapa 2).
2.  **Cálculo de Confiabilidade (IRR):** Aplicação do **Kappa de Cohen**.
    *   **IRR < 0,80:** O item é sinalizado para "Redesenho Estrutural" (ambiguidade excessiva).
3.  **Sessão de Harmonização:** Discussão focada nas evidências da Situação-Problema (SP) para resolver divergências.
4.  **Trilha de Auditoria:** Registro formal das justificativas para cada classificação final.

## Exemplo de Aplicação
No **Item 99 (Biologia)**, a classificação "Forte" depende da confirmação, via triangulação, de que o verbo "verificar" exige domínio do ciclo biológico (avaliação de riscos) e não apenas memória.
